📌 Project Overview

This project is based on the Digital Literacy course where I worked as a Student Digital Ambassador. The main aim of this project was to understand how to use digital tools effectively, communicate professionally, and stay safe online.
I completed five different tasks related to digital awareness, online platforms, communication, and cyber safety. All the work is organized in this repository.
________________________________________
📂 Tasks Summary

🔹 Task 1 – Digital Literacy Infographic
Created an infographic using Canva explaining digital literacy, useful tools, and its importance in daily life.

🔹 Task 2 – Digital Portfolio
Set up professional profiles on platforms like GitHub, LinkedIn, and Kaggle and understood their use for future career growth.

🔹 Task 3 – Platforms Exploration
Solved a basic coding problem on HackerRank and created a Google Form quiz to understand collaboration tools.

🔹 Task 4 – Email Etiquette
Wrote two professional emails and created a checklist for proper social media usage.

🔹 Task 5 – Cybercrime Awareness
Worked on a cybercrime case study and created a prevention checklist to stay safe online.
